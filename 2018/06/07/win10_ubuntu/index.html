安装 win10 子系统 Ubuntu

  • 设置 → 更新和安全 → 开发者选项 → 使用开发人员功能 → 勾选「开发人员模式」
  • 控制面板 → 程序和功能 → 启用或关闭 Windows 功能 → 勾选「适用于 Linux 的 Windows 子系统」
  • 重启系统
  • 打开 Windows Stroe，下载 Ubuntu 即可
